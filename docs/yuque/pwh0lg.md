# 5. 输入短语和自定义的输入出

如下是 sj 的快速输入，把下面贴到短语里面，然后选择“添加”即可。

```
// sj的
%yyyy%-%MM%-%dd% %HH%:%mm%:%ss%

//rq的
%yyyy%年%MM%月%dd%日 %HH%:%mm%:%ss%
```

然后 你就可以看到如下界面：
